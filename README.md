# ♟ Smart Chess Learning Board

## 📌 Project Overview
Smart Chess Learning Board is an intelligent Arduino-based embedded system designed to help beginners learn chess interactively.  

The system detects piece movement, validates legal moves, and provides visual feedback using LEDs to guide players during gameplay.

This project combines embedded systems, hardware interfacing, and logical validation of chess rules.

---

## 🎯 Objectives
- Assist beginners in learning legal chess moves
- Detect piece lifting and placement
- Indicate illegal moves using LED signals
- Provide check/checkmate indication
- Improve learning through interactive feedback

---

## 🛠 Hardware Components Used
- Arduino Microcontroller
- LED Matrix / Indicator LEDs
- Sensors ( Reed Switch)
- Chess Board with grid mapping
- Power Supply Module
- Connecting Wires

---

## 💻 Software Used
- Arduino IDE
- Embedded C Programming
- HTML

---

## ⚙️ Working Principle
1. Each chess square is monitored using sensors.
2. When a piece is lifted, the system detects the change.
3. The software validates the move based on chess rules.
4. Legal moves are allowed.
5. Illegal moves trigger LED indication.
6. System can indicate special conditions like check or checkmate.
